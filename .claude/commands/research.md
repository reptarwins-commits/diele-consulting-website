# Save Research / Finding

We just figured something out worth keeping. Save it so we don't have to
re-discover it later.

1. Create a `docs/notes/` folder if it doesn't exist.
2. Write a short Markdown file at `docs/notes/YYYY-MM-DD_short-title.md`
   capturing: the question, what we found, and why it matters.
3. Keep it concise — a few paragraphs, not an essay.
4. Tell me the file path you created.
